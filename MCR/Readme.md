# Платформа онлайн-курсів - База даних

Проєктування бази даних для платформи онлайн-навчання, де викладачі створюють курси, студенти реєструються, а система відстежує завершення курсів та прогрес через уроки

---

## Схема бази даних

### Таблиці

| Таблиця | Опис |
|---|---|
| `instructors` | Викладачі - облікові дані, біографія, спеціалізація |
| `students` | Студенти - контактна інформація, дата реєстрації |
| `courses` | Курси - назва, опис, ціна, категорія |
| `lessons` | Уроки - належать до курсу, впорядкована послідовність |
| `enrollments` | Реєстрації - зв'язок студент↔курс зі статусом |
| `lesson_progress` | Прогрес студента по кожному уроку |

## OLTP запити

### INSERT
- Додавання нового студента
- Реєстрація студента на курс

### UPDATE
- Позначення реєстрації як завершеної
- Зміна ціни курсу

### DELETE
- Видалення реєстрації студента
- Видалення студента

### SELECT
- Знайти всі курси що викладаються конкретним викладачем (пошук за іменем)
- Знайти всіх студентів що відвідували конкретний курс

---

## OLAP запити

| Запит | Опис |
|---|---|
| Завершення за категоріями | Середній відсоток завершення курсів у розрізі категорій |
| Топ-5 викладачів | Рейтинг викладачів за загальною кількістю зареєстрованих студентів |
| Ранжування курсів | Курси ранжовані за кількістю реєстрацій у межах кожної категорії (віконна функція `RANK`) |
| Незавершені студенти | Студенти що зареєструвалися на кілька курсів але не завершили жодного (CTE) |
